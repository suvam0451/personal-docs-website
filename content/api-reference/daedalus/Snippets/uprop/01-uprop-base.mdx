## uprop base

All base uprop macros have **VisibleAnywhere** and **BlueprintReadWrite** tags
These are only useful in header files

The parameter is always set to Parameters

![base uprop](../../../../images/DaedalusAPI/introduction-module/007.gif)

### list of base uprop snippets

---

- `uprop` : Empty. Completely upto user
- `uprop_int`
- `uprop_bool`
- `uprop_float`
- `uprop_vector`
- `uprop_transform`
- `uprop_ismc` : Instanced static mesh component
- `uprop_mesh` : StaticMesh (not component)
- `uprop_meshcomp` : StaticMeshComponent
- `uprop_switch` : concept of boolean switches
- `uprop_ai_sightconfig` : UAISenseConfig_Sight
- `uprop_sphere` : SphereComponent
- `uprop_box` : BoxComponent
- `uprop_abilitysyscomp` : Ability System component
- `uprop_spline` : SplineComponent

### Special cases

- `uprop_timer` : Generates delegate + handle
